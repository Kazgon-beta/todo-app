# Simple　ToDo　App

ブラウザで動作するシンプルなToDoアプリです。
タスクの追加、削除ができ、**localStrage**によってブラウザを閉じてもタスクが保存されます。

---
## 機能一覧

|機能|説明|
|:-----|-----:|
|タスクの追加|入力欄にテキストを入力し「追加」ボタンを押すとリストに追加できます。|
|タスクの削除|タスク右側のxを押すと、削除できます。|
|ストレージに保存|ブラウザの読み込みを行ったり、タブを閉じてもタスクを保存できます。|
